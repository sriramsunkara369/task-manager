# Task Manager Web Application

A feature-rich Task Manager built using HTML, CSS and JavaScript.

## Features
- User Authentication (Login & Create Account)
- Profile Management (Change username, password, profile picture)
- Task Management (Add, Edit, Delete, Mark as Completed)
- Password visibility & strong password suggestion
- Settings with privacy policy, help, credits
- Permanent account deletion
- Clean, responsive UI

## Technologies Used
- HTML5
- CSS3
- JavaScript (LocalStorage)

## How to Run
1. Clone the repository
2. Open `login.html` in a browser
